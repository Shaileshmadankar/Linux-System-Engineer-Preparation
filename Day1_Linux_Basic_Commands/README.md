# Day 1 — Linux Basic Commands

Today I practiced basic Linux commands.

## 1. pwd
Shows the current working directory.

Output:
/home/shailesh

## 2. ls
Lists files and directories.

Output:
Documents  Downloads  Music snap 

## 3. ls -l
Shows detailed list of files.

Output:
drwxr-xr-x 2 joy joy 4096 Mar 10 Documents

## 4. clear
Clears the terminal screen.

Output: 
clear entire screen


## 5. date
Shows current system date and time.

Output:
Thu Mar 13 10:20:15 IST 2026

## 6. whoami
Shows the current logged in user.

Output:
shailesh

## 7. history
Shows recent execute commands.

Output:
1 pwd
2 ls
3 whoami
