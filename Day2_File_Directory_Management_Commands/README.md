# Day 2 — File & Directory management

Today I practiced file and directory
management commands in linux.

## 1. touch
Create a new empty file.
Ex. touch system.txt


## 2. mkdir
Create a new directory.
Ex. mkdir test_folder


## 3. rm
Delete a file.
Ex. rm file.txt

## 4. rmdir
Delete an empty directory.
Ex. rmdir test_folder


## 5. cp
Copy files.
Ex. cp filenew.txt file2.txt


## 6. mv
Move or rename files.
Ex. file3.txt newfile2.txt
