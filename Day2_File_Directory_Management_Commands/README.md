# Day 2 — File & Directory Management

Today I practiced file directory management commands
in linux.

1. touch
Create a new empty file.
Ex. touch system.txt

2. mkdir
Create a new directory.
Ex. mkdir test

3. rm
Delete a file
Ex. rm result

4. rmdir
Delete an empty directory.
Ex. rmdir test

5. cp
Copy files.
Ex. cp filenew.txt file2.txt
